# project-overview.md

## 🎯 프로젝트 개요
- **프로젝트명**: SPOT 스마트 가이드
- **타입**: QR 기반 위치 정보 서비스 (PWA 웹앱)
- **핵심 기능**: QR 스캔 → 해당 위치(SPOT) 정보 제공
- **사용자**: 관광객, 지역 방문객

## 📱 화면 구조
### 1. 진입 플로우 (3개 화면)
- QR 스캔 페이지
- 스플래시 화면
- SPOT 정보 화면 (예: 35번 가로등)

### 2. 메인 화면 + 5개 탭
- 메인 화면 (카드형 레이아웃)
  - 탭1: 부평 이야기 (카테고리별 보기)
  - 탭2: 주변 편의시설 (지도)
  - 탭3: 행사 정보 안내
  - 탭4: 내 스탬프 현황

### 3. 카테고리 상세 (4개 ZONE)
- A ZONE: 거리 소개
- B ZONE: 인물
- C ZONE: 행사/축제  
- D ZONE: 역사

## 🔄 사용자 플로우
1. QR 코드 스캔
2. 해당 SPOT 정보 확인
3. 메인 화면 진입
4. 카테고리별 콘텐츠 탐색
5. 스탬프 수집
